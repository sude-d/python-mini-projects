# 📸 Instagram Automation Bot

A Python-based automation tool developed using the **Selenium** library. This project is designed to automate basic Instagram interactions such as logging in and managing follower lists.

---

## 🚀 Features
* **Automated Login:** Securely logs into your account using credentials.
* **Follower Management:** Built-in functions to handle following/unfollowing tasks.
* **Browser Support:** Optimized for Safari and Chrome drivers on macOS.
