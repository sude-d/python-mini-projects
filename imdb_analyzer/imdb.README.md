# 🎬 IMDb Top Chart Scraper

A specialized Python tool to extract movie data from IMDb's "Top Rated" lists.

## 🌟 Key Features
* Extracts **Movie Titles** and **Ratings**.
* Scrapes **Release Years** and **Director Info** (if applicable).
* Saves data in a clean, readable format for further analysis.

## 🛠️ Built With
* **Python 3.13**
* **BeautifulSoup4:** For HTML parsing and data extraction.
* **Requests:** For making secure connections to IMDb servers.

## 💡 Why this project?
This project demonstrates my ability to handle large-scale web structures and extract specific information from high-traffic websites. It's a stepping stone for my future **Natural Language Processing (NLP)** projects involving movie reviews.
