# Harcama Hesaplayıcı

Bu proje, Python ile yazdığım basit bir harcama hesaplayıcı programıdır.

## Özellikler
- Kullanıcıdan sınırsız sayıda gider girişi alır
- 'q' girildiğinde program sonlanır
- Geçersiz girişlerde uyarı verir

## Örnek Çalışma
Girdi:
50
30
20
q

Çıktı:
Toplam harcama: 100.0

## Kullanım
Terminalde proje klasörüne gidip:

python3 harcama_hesaplayıcı.py

komutunu çalıştırın.
