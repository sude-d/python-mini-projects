# 🏦 Simple ATM Application

This is a basic **Python** project that simulates an Automated Teller Machine (ATM) experience. It helps users manage their bank accounts through a command-line interface.

## 🌟 Features
* **Balance Inquiry:** Check your current account balance anytime.
* **Cash Withdrawal:** Withdraw money with balance checks.
* **Deposit:** Add money to your account instantly.
* **Exit:** Securely end your session.

## 🛠️ How to Run
1. Make sure you have **Python 3.x** installed.
2. Clone this repository or download the `bankamatik.py` file.
3. Open your terminal and run:
   ```bash
   python bankamatik.py
